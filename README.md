## Features

- **Responsive Design**: Optimized layouts for mobile (375px), tablet, and desktop (1440px) screens using CSS media queries.
- **Interactive Elements**: Hover effects on the register button for enhanced user experience.
- **Modern Typography**: Utilizes Poppins for headings and Open Sans for body text.
- **Accessible**: Follows WCAG guidelines for better accessibility.

## Technologies Used

- **HTML5**: Semantic markup for structure.
- **CSS3**: Custom properties (variables), Flexbox for layout, and responsive design with media queries.
- **Fonts**: Google Fonts (Poppins and Open Sans).
- **Tools**: Mobile-first workflow approach.

## Installation and Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Sana-zen/huddle-landing-page.git
   ```
2. Navigate to the project directory:
   ```bash
   cd huddle-landing-page
   ```
3. Open `index.html` in your preferred web browser to view the landing page.
